# Mergington High School Activities

A web application for managing extracurricular activities at Mergington High School. The platform allows students to browse activities and teachers to manage student registrations.

## Features

### For Students
- Browse all available extracurricular activities with detailed information
- View activity schedules, descriptions, and available spots
- Filter activities by:
  - Category (Arts, Sports, Academic, Technology, Music, Special Interest)
  - Day of the week
  - Time of day (Morning, Afternoon, Evening)
  - Difficulty level (Beginner, Intermediate, Advanced, All Levels)
- Search activities by name or description
- View activity capacity and current enrollment

### For Teachers and Administrators
- Secure login system with role-based access
- Register students for activities
- Unregister students from activities
- Manage activity enrollments

### Available Activities
The platform includes a variety of activities such as:
- Chess Club
- Web Development
- Robotics Club
- Basketball Team
- Drama Club
- Math Olympiad
- Music Band
- Photography
- Debate Team
- Soccer Team
- Art Studio
- Science Fair Prep
- Manga Maniacs

## Technical Features
- Teacher authentication and session management
- Real-time activity filtering and search
- Dark mode support
- Responsive design
- MongoDB database for data persistence
